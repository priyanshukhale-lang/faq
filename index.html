          How to get your Refresh Token (one-time setup):

          1. api-console.zoho.in → Self Client → Generate Code

          2. Scope: ZohoCRM.modules.leads.READ,ZohoCRM.modules.leads.WRITE · Duration: 10 minutes

          3. Exchange via: POST https://accounts.zoho.in/oauth/v2/token with grant_type=authorization_code&code=YOUR_CODE&client_id=...&client_secret=...&redirect_uri=https://localhost

          4. Save the refresh_token from the response — it never expires unless revoked
